# Developer Guide

Extend AgencyOS functionality.

## Webhook Setup

Receive real-time updates for events.

1. Go to **Developers > Webhooks**
2. Click **Add Endpoint**
3. Enter your URL and select events
4. Verify signature using your secret

## SDK Usage

We provide SDKs for:
- [Node.js](https://npmjs.com/package/agencyos-sdk)
- [Python](https://pypi.org/project/agencyos-sdk)
