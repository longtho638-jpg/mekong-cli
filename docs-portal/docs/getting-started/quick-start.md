# Quick Start Guide

This guide will help you build your first agency workflow.

## 1. Create a Project

Navigate to the **Projects** dashboard and click "New Project".

1. Enter project name: `My First Campaign`
2. Select template: `Marketing Launch`
3. Click **Create**

## 2. Invite Team Members

Go to **Settings > Team** and invite your colleagues.

- **Admin**: Full access
- **Editor**: Can create and edit content
- **Viewer**: Read-only access

## 3. Configure Integrations

AgencyOS connects with your favorite tools:

- **Slack**: For notifications
- **Stripe**: For payments
- **GitHub**: For code deployment

Go to **Integrations** tab to connect.
