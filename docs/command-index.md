## Quick Reference by Use Case

### üéØ Getting Started

**First-time setup**:
```bash
mekong init "Your Agency Name"        # Initialize agency
mekong setup-vibe --location "City"   # Configure environment
```

**Modern equivalent**:
```bash
cc client init "Your Agency Name"
cc agent configure-vibe --location "City" --tone "Professional"
```

### üíº Client Workflow

**From lead to contract**:
```bash
# Step 1: Add lead

# Step 2: Generate quote
cc sales quote "Client Name" --budget 5000

# Step 3: Validate alignment
/win3 --deal "Client Name"

# Step 4: Generate proposal
cc sales proposal "Client Name"

# Step 5: Create contract
cc sales contract "Client Name"

# Step 6: Onboard client
cc client add "Client Name" --status active
```
