## 🏛️ ĐIỀU 1: ĐỊNH NGHĨA R = ROI = RaaS

1. **RaaS không chỉ là Robot**: RaaS chính xác là **ROI as a Service**. Chúng ta bán kết quả, không bán sức lao động.
2. **Sứ mệnh**: Giúp Developer hoàn thành task trong 15-60 phút (Engineering ROI) và giúp Non-tech User tạo ra tiền trên UI (Operational ROI).
3. **Mục tiêu**: Biến 100% chi phí tech thành 1000% lợi nhuận ROI cho khách hàng.

---

## 💰 ĐIỀU 2: CHIẾN LƯỢC DÒNG TIỀN KÉP (DUAL-STREAM)

Mọi dự án thuộc hệ sinh thái AgencyOS BẮT BUỘC triển khai 2 dòng doanh thu:

### 1. Dòng Thu Công Nghệ (Engineering ROI / Dev Key)
- **Target**: Developers, AI Engs, Agency Owners.
- **Giá trị**: Tốc độ (Speed), Tự động hóa (Automation), Hệ thống (CTO Auto-Pilot).
- **Gate**: `RAAS_LICENSE_KEY` để mở khóa CLI commands, Premium Agents, Deep Intelligence.
- **Slogan**: "Build faster. Code smarter. Fix instantly."

### 2. Dòng Thu Vận Hành (Operational ROI / User UI)
- **Target**: Business Owners, Marketers, End-users.
- **Giá trị**: Kết quả (Results), Thu nhập (Revenue), Trải nghiệm (Premium UI).
- **Gate**: Subscription/Pay-per-use ngay trên Web UI để truy cập Dashboard, Signals, Marketing Factory.
- **Slogan**: "Zero setup. Real money. High ROI."

## 🧬 ĐIỀU 3: CẤU TRÚC HƯ — THỰC (BINH PHÁP CH.6)

Tuân thủ triệt để chiến lược Open Source để chiếm lĩnh thị trường:

| Loại | Thành Phần | Trạng Thái | ROI Phục Vụ |
|:-----|:-----------|:-----------|:------------|
| **HƯ (Open)** | Source code, Base Patterns, CLI Core | **Public GitHub** | Thu hút cộng đồng, Viral, Free CI/CD |
| **THỰC (Closed)** | AI Brain (Opus), Data, Prod Infra, Keys | **Gated / Private** | Kiếm tiền (Monetization), Bảo mật |

---

## ⚖️ ĐIỀU 4: QUY TẮC PHÁT TRIỂN BẮT BUỘC

1. **Rule of 60**: Một mission/feature của Robot phải ship được giá trị trong tối đa 60 phút.
2. **UI "WOW" Factor**: UI không chỉ để nhìn, UI phải để chốt đơn (Conversion-First).
3. **License-First**: Không code tính năng premium mà không có license gate tương ứng.
4. **Learn from Teacher**: Mọi dự án mới phải copy và tiến hóa từ pattern của "Teacher Project" thành công nhất (Hiện tại là `algo-trader`).
