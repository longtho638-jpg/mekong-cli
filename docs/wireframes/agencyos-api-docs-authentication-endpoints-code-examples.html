API Reference

v1.0 — Stable

The AgencyOS REST API lets you create tasks, manage agents, and receive webhook events programmatically. All requests must be authenticated using a Bearer token.

Base URL
https://api.agencyos.ai/v1

Authentication

Authenticate all API calls by including your API key in the Authorization header. Generate API keys from the Dashboard → API Keys.

Request 
GET /v1/tasks

Authorization: Bearer ao_live_sk_••••••••••••••••
Content-Type: application/json
X-Org-ID: org_01HFYZ3K8M
Response 200 OK 
{
  "data": [...],
  "meta": {
    "total": 47,
    "page":  1,
    "limit": 20
  }
}
POST /v1/tasks

Create Task

Submit a new task for an agent to execute. Returns immediately with a task ID. Poll /v1/tasks/{id} or use webhooks for status updates.

Request Body 
{
  "goal":   "Audit codebase for TS issues",
  "agent":  "tester",
  "mode":   "auto",
  "notify": {
    "webhook": "https://your.app/hook"
  },
  "meta": {
    "ref": "sprint-42"
  }
}
Response 201 Created 
{
  "id":        "task_01HK9Z",
  "status":    "queued",
  "goal":      "Audit codebase...",
  "agent":     "tester",
  "mcu":       null,
  "created_at":"2026-03-02T09:42:00Z"
}
Body Parameters
Param Type Required Description
goal string Required Plain-English description of the task to execute.
agent string Optional Agent ID to use. Defaults to auto (auto-select).
mode string Optional auto | plan | execute. Default: auto.
notify object Optional Webhook URL to notify on task completion or failure.

Webhooks

AgencyOS sends POST requests to your endpoint for async task events. Verify authenticity using the X-AgencyOS-Signature header (HMAC-SHA256).

task.completed

Fired when a task finishes and passes all quality gates.

task.failed

Fired when a task exhausts all self-healing retries.

task.step_complete

Fired after each verified step in a multi-step recipe.

agent.spawned

Fired when a new agent instance is spawned for a task.

Webhook Payload task.completed 
{
  "event":     "task.completed",
  "timestamp": "2026-03-02T09:44:18Z",
  "data": {
    "task_id":   "task_01HK9Z",
    "mcu_used": 42,
    "verified": true,
    "result":   { "summary": "0 type errors" }
  },
  "org_id":    "org_01HFYZ3K8M"
}

SDKs

Official SDKs wrap the REST API with typed clients, retry logic, and webhook helpers.

Install 
# npm
npm install @agencyos/sdk

# pnpm
pnpm add @agencyos/sdk
Usage — TypeScript 
// Initialize client
import { AgencyOS } from '@agencyos/sdk';

const client = new AgencyOS({
  apiKey: process.env.AGENCYOS_API_KEY,
  orgId:  process.env.AGENCYOS_ORG_ID,
});

// Create a task
const task = await client.tasks.create({
  goal:  "Audit codebase for TS issues",
  agent: "tester",
});

// Poll until done
const result = await client.tasks.wait(task.id);
console.log(result.verified); // true

Error Codes

Code HTTP Description
unauthorized 401 Missing or invalid API key.
forbidden 403 Key lacks permission for this action.
quota_exceeded 429 MCU quota exhausted. Upgrade plan or wait for reset.
task_not_found 404 Task ID does not exist or belongs to another org.
internal_error 500 Unexpected server error. Check status.agencyos.ai.
