# Skills Overview

AgencyOS includes 200+ specialized skills that extend agent capabilities.

## Skill Categories

### Development

- `frontend-design` - UI/UX implementation
- `backend-development` - API and database
- `testing-patterns` - Test strategies

### Business

- `copywriting` - Marketing copy
- `seo-fundamentals` - Search optimization
- `pricing-strategy` - Monetization

### AI/ML

- `rag-implementation` - Vector search
- `prompt-engineering` - Prompt design
- `llm-app-patterns` - AI application patterns

### Security

- `security-review` - Security auditing
- `vulnerability-scanner` - Threat detection

## Using Skills

Skills are automatically activated based on context:

```bash
# This activates frontend-design skill
/cook create landing page with modern glassmorphism UI
```

See [Skills Catalog](../skills-catalog-report.md) for the complete list.
