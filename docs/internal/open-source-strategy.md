## Principle: Open = How, Closed = Where + What

- **HOW** to build autonomous agents → Open (mekong-cli)
- **WHERE** to run at scale → Closed (Antigravity + OpenClaw)
- **WHAT** results are delivered → Closed (AgencyCoins gated)
