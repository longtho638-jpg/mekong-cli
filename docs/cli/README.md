# CLI Usage

AgencyOS CLI (Claude Code) provides a unified command interface for all business operations.

## Installation

```bash
npm install -g @anthropic/claude-code
```

## Quick Start

```bash
# Start the CLI
cc

# Run a command
/cook add login feature with social auth

# Get help
/help
```

## Core Commands

- `/code` - Build features, write code
- `/test` - Run tests
- `/ship` - Deploy to production
- `/scout` - Analyze codebase
- `/review` - Code review

See [CLI Reference](../CLI_REFERENCE.md) for complete documentation.
