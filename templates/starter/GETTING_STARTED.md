## Prerequisites

- Python 3.8+ installed
- A valid AgencyOS license key

---

## Step 1: Activate Your License

Run the activation script:

```bash
python activate.py YOUR-LICENSE-KEY
```

Example:
```bash
python activate.py AGENCYOS-ABCD-1234-WXYZ
```

You should see:
```
‚úÖ LICENSE ACTIVATED SUCCESSFULLY!
```

---

## Step 2: Run Your First Command

Try the help command to see all available commands:

```
/help
```

---

## Step 3: Explore Commands

### Marketing
```
/marketing-plan       # Generate marketing plan
/content-calendar     # Create content calendar
/social-post          # Generate social media posts
```

### Sales
```
/proposal             # Create sales proposal
/pitch-deck           # Generate pitch deck
/crm-sync             # Sync with CRM
```

### Finance
```
/invoice              # Generate invoice
/runway-calc          # Calculate runway
/expense-report       # Create expense report
```

### Operations
```
/sop-gen              # Generate SOPs
/workflow             # Create workflows
/meeting-notes        # Summarize meetings
```
