# Co-founder Profile — Mekong CLI

**Status:** Active search | **Timeline:** Q2 2026 if traction confirmed

---

## Do We Need a Co-founder?

**Short answer:** Not immediately. Solo is viable to $10K MRR.
**When to add:** If growth stalls at $5K MRR due to bandwidth, or if raising pre-seed (investors prefer 2+ founders).

---

## Ideal Co-founder Profile

### Role: Technical CTO / AI Infrastructure

**What we need (not what we want):**

| Need | Why |
|------|-----|
| Python backend depth | Core PEV engine is Python; needs hardening |
| LLM infrastructure experience | Prompt engineering, routing, cost optimization at scale |
| DevOps / Cloudflare Workers | Our zero-cost infra stack needs an owner |
| OSS community building | GitHub presence, PR reviews, contributor culture |
| Async work ethic | Solo founder rhythm; no office, no standups |

**What we do NOT need:**
- Another generalist CEO
- Someone who wants to rebuild in Go/Rust immediately
- VC-obsessed mindset before PMF
- Anyone who can't ship solo

## Equity Framework

| Scenario | Founder | Co-founder | Cliff | Vest |
|----------|---------|-----------|-------|------|
| Pre-revenue join | 60% | 40% | 1yr | 4yr |
| Post $5K MRR | 70% | 30% | 1yr | 4yr |
| Post $10K MRR | 75–80% | 20–25% | 6mo | 3yr |

Equal split (50/50) only if co-founder brings equivalent prior work or significant capital.

| Channel | Approach |
|---------|----------|
| GitHub contributors | Watch who forks and opens PRs; invite top contributors |
| Hacker News "Who wants to be hired" | Monthly thread; filter for AI + Python |
| Twitter/X AI dev community | Build in public, attract alignment |
| YC co-founder matching | Apply even without intent to do YC |

---

## Interview Process

1. **Async intro:** 500-word email — what they've built, what they want
2. **Code review:** Review a PR or architecture decision in Mekong codebase
3. **Paid test project:** 1-week paid project ($500–$1K), ship a real feature
4. **Values conversation:** Binh Phap philosophy, remote work, equity expectations
5. **Reference check:** Talk to 2 prior collaborators

---

## Red Flags

- Wants to redesign architecture before understanding current one
- Talks about valuation before product
- Needs external validation to make decisions
- Can't work without a manager
- Hasn't shipped anything publicly

---

## Alternative: First Hire (Not Co-founder)

If no co-founder found by $5K MRR, hire:
- **Contract DevRel** ($2K/mo): Handles community, docs, tutorials
- **Contract Python engineer** ($3K/mo, 20hr/week): API hardening, test coverage

Equity: 0.25–0.5% vested over 2 years for early contractors.
