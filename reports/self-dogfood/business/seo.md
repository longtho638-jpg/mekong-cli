## On-Page SEO Checklist

### Homepage / Docs Site
- [ ] `<title>`: "Mekong CLI — AI Business Platform | 289 Commands, Any LLM"
- [ ] Meta description: "Open source AI CLI that runs your company. 289 commands across 5 business layers. Works with any LLM. Free to install."
- [ ] H1: Primary keyword present
- [ ] Schema markup: SoftwareApplication
- [ ] Open Graph tags for social sharing
- [ ] Canonical URL set

### GitHub README (acts as SEO landing page)
- [ ] First 160 chars contain primary keyword
- [ ] GitHub Topics: `ai`, `cli`, `llm`, `business-automation`, `python`, `openai`, `anthropic`
- [ ] README has demo GIF (increases time-on-page)

### Dev.to / Hashnode Articles
- [ ] Each article targets one keyword cluster
- [ ] Internal links between articles
- [ ] Links back to GitHub repo (authority signals)
