## Market Trends (2026)

### Tailwinds
1. **"Vibe coding" normalization** — more non-devs using CLI tools as AI lowers barrier
2. **LLM cost collapse** — GPT-4 class models now at $0.002/1K tokens; economics improve monthly
3. **OSS trust premium** — post-OpenAI drama, dev community increasingly values open source
4. **Solo founder boom** — bootstrapped companies growing 40% YoY on Indie Hackers
5. **Anti-SaaS fatigue** — "SaaS sprawl" is a real pain point; consolidation tools win

### Headwinds
1. **OpenAI Operator** — if it goes mainstream, could commoditize agent platforms
2. **Claude Code / GitHub Copilot Workspace** — big tech expanding into adjacent territory
3. **LLM API key management** — BYOK model shifts responsibility to users (support burden)
4. **OSS forks** — anyone can fork and self-host without paying
