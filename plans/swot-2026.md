# Mekong CLI — SWOT Analysis 2026

## Strengths
- **PEV Engine unique**: Plan-Execute-Verify với auto-rollback — không tool nào có
- **Universal LLM**: 3 vars, bất kỳ provider — không lock-in
- **5-tầng doanh nghiệp**: Founder→Business→Product→Engineer→Ops — scope rộng nhất thị trường
- **Open source + RaaS**: Core miễn phí, monetize qua credits
- **176 machine contracts**: Factory validation đảm bảo quality ở scale
- **245 commands + 245 skills**: Coverage lớn hơn bất kỳ CLI agent nào

## Weaknesses
- **Single maintainer**: Bus factor = 1, chưa có community
- **Docs outdated**: README/CLAUDE.md patch nhiều lần, không coherent
- **No paying customer yet**: Revenue = $0
- **Python-only CLI**: Không có web dashboard cho non-tech users
- **No npm/PyPI publish**: Chưa `pip install mekong-cli` được từ registry

## Opportunities
- **AI agent market exploding**: 2026 là năm agents mainstream
- **Founder tools gap**: Không tool nào cover idea→IPO journey
- **OpenRouter/Ollama adoption**: Users muốn LLM flexibility
- **Fork economy**: Mỗi fork = potential RaaS customer
- **Vietnam tech ecosystem**: First-mover cho AI agency platform tiếng Việt

## Threats
- **Claude Code / Cursor**: IDE-native agents với UX tốt hơn CLI
- **Devin / Cognition**: Full autonomous dev agents, well-funded
- **Aider**: Established open-source CLI agent
- **LLM cost race to zero**: Credits model bị compress margins
- **Attention span**: Users muốn GUI, không muốn learn CLI commands
