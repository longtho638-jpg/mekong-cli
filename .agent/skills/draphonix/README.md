# Draphonix Skills

This skill implements multi-agent coordination patterns inspired by [Draphonix](https://github.com/draphonix/skills).

## Features

- **Orchestrator Pattern**: Manage complex workflows.
- **Planning**: Break down tasks into beads/tracks.
- **Knowledge**: Extract and store learnings.

## Usage

Designed to be used with the `amp` cli or as a reference implementation for multi-agent logic.

## Components

- `planning`: Execution plans
- `orchestrator`: Worker coordination
- `worker`: Task execution
