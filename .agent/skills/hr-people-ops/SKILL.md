# HR & People Ops Skill

Build modern people operations with HRIS automation, hiring pipelines, and employee experience systems.

## When to Use

- Setting up HRIS (Human Resource Information System)
- Designing recruiting and hiring workflows
- Building onboarding programs (30-60-90 day plans)
- Performance review systems and feedback loops
- Compensation planning and benchmarking
- Employee engagement surveys and analysis
- Org chart and team structure design
- Payroll and benefits administration
- Compliance (labor laws, I-9, EEO)
- Offboarding and knowledge transfer

## Hiring Pipeline

```
Sourcing → Screening → Interview → Assessment → Offer → Onboarding
   ↓           ↓           ↓           ↓          ↓         ↓
LinkedIn    ATS filter   Structured  Take-home   Comp     30-60-90
Referrals   Resume AI    Scorecards  Live coding  Band     Buddy system
Job boards  Phone screen Panel       Culture fit  Benefits Feedback
```

## Onboarding Template (30-60-90)

```markdown
## Week 1: Foundation
- [ ] IT setup (laptop, accounts, access)
- [ ] Meet manager, buddy, key stakeholders
- [ ] Read product/company docs
- [ ] Complete compliance training

## Day 1-30: Learn
- [ ] Shadow team members on 3+ projects
- [ ] Complete role-specific training modules
- [ ] Deliver first small contribution
- [ ] Weekly 1:1 with manager

## Day 31-60: Contribute
- [ ] Own 1-2 tasks independently
- [ ] Present learnings to team
- [ ] Give/receive peer feedback
- [ ] Mid-point check-in with manager

## Day 61-90: Own
- [ ] Lead a project or initiative
- [ ] Mentor newer team member (if applicable)
- [ ] Set quarterly goals (OKRs)
- [ ] 90-day review with manager
```

## Key Metrics

| Metric | Target | Purpose |
|--------|--------|---------|
| Time to Hire | < 30 days | Recruiting efficiency |
| Offer Acceptance Rate | > 85% | Comp competitiveness |
| 90-day Retention | > 90% | Onboarding quality |
| eNPS | > 30 | Employee engagement |
| Voluntary Turnover | < 15%/year | Retention health |
| Training Completion | > 95% | Compliance + development |

## Key Best Practices (2026)

**Structured Interviews:** Same questions + scorecard for every candidate — reduces bias 40%
**Async-First Onboarding:** Self-paced learning modules, recorded walkthroughs, written docs
**Continuous Feedback:** Weekly/bi-weekly 1:1s, quarterly reviews, real-time recognition
**Compensation Transparency:** Published bands, clear leveling frameworks
**Global-Ready:** Deel/Remote.com for international hiring without local entities
