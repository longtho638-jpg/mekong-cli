Multimodal image analysis with Google Gemini.

## When to Use

- Screenshot analysis
- UI/UX review from images
- Document/diagram understanding

## Capabilities

1. **UI Analysis** - Identify components, layout, interactions
2. **Document OCR** - Extract text from images
3. **Diagram Understanding** - Parse flowcharts, architecture
4. **Design Review** - Evaluate aesthetics, accessibility
5. **Code from Screenshots** - Recreate UI from images

## Usage Pattern

```
# Attach image and say:
"Use gemini-vision to analyze this screenshot"
"Use gemini-vision to extract text from this image"
"Use gemini-vision to recreate this UI in React"
```

## Output Format

For UI analysis:
- **Layout** - Structure and hierarchy
- **Components** - Identified UI elements
- **Colors** - Palette used
- **Typography** - Fonts and sizes
- **Interactions** - Implied behaviors
