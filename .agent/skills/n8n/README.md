# n8n Skill

This skill integrates [n8n](https://n8n.io) workflow automation into the AgencyOS system.

## Features

- **Workflow Automation**: Connects over 400 services.
- **AI Integration**: Native LangChain nodes for building AI agents.
- **Self-Hosted**: Run locally or on your own infrastructure.

## Usage

You can run n8n using `npx`:

```bash
npx n8n
```

This will start the n8n server on port 5678.

## Integration

To integrate with AgencyOS, use Webhooks in n8n to trigger workflows from your agents, or use the n8n API to manage workflows programmatically.

## Requirements

- Node.js installed
- `npx` available
