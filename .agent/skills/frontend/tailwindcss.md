## When to Use

- Styling components with utilities
- Creating responsive layouts
- Implementing dark mode
- Custom design systems

## Tech Stack

- **Tailwind**: v4 (native CSS)
- **Plugins**: Typography, Forms, Container Queries
- **Dark Mode**: Class-based strategy
- **Responsive**: Mobile-first breakpoints

## Key Patterns

1. **Utility-First** - Compose utilities, avoid custom CSS
2. **Responsive** - sm:, md:, lg:, xl:, 2xl:
3. **Dark Mode** - dark: variant
4. **States** - hover:, focus:, active:
5. **Groups** - group-hover:, peer modifiers
