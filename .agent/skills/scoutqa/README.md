# ScoutQA Skill

This skill integrates [ScoutQA](https://scoutqa.ai), an AI-powered QA testing platform for "vibe coding" applications.

## Features

- **Autonomous Testing**: AI agents explore and test your app.
- **No-Code Setup**: Just provide a URL.
- **Visual Reports**: Screenshots and video recordings of bugs.

## Usage

This skill acts as a documentation and configuration point for ScoutQA integration.

### Setup

1. Sign up at [scoutqa.ai](https://scoutqa.ai).
2. Configure your AgencyOS URL in the ScoutQA dashboard.
3. Review reports in the dashboard or via email.

## Integration

Future versions may integrate via API to trigger tests automatically on deployment.
