# ⚙️ Backend Development Skill

Expert backend development with modern Python and Node.js patterns.

## When to Use

- Building REST/GraphQL APIs
- Database schema design
- Authentication/authorization
- Server-side business logic

## Tech Stack

- **Python**: FastAPI, Pydantic, SQLAlchemy
- **Node.js**: Express, Next.js API Routes
- **Database**: PostgreSQL, Supabase, Redis
- **Auth**: Supabase Auth, JWT
- **Deploy**: Vercel, Cloud Run, Docker

## Best Practices

1. **Type Safety** - Pydantic models, TypeScript
2. **Error Handling** - Structured exceptions
3. **Logging** - Structured logging with context
4. **Testing** - Unit + integration tests
5. **Documentation** - OpenAPI auto-generation

```
"Use backend-development to create user CRUD API"

```

## Output Standards

- RESTful conventions
- Proper HTTP status codes
- Input validation
- Error response schema
