## Design Principles

1. **Form Follows Function** - Design for the user's goal
2. **Consistency** - Use established patterns
3. **Accessibility** - WCAG AA minimum
4. **Delight** - Add micro-interactions and polish
5. **Performance** - Lightweight, fast-loading

## Output Format

For design tasks:
1. **Context** - User goals and constraints
2. **Wireframe** - Structure and layout (ASCII or description)
3. **Specs** - Colors, spacing, typography tokens
4. **Interactions** - Hover, click, transition states
5. **Accessibility** - ARIA labels, contrast ratios
