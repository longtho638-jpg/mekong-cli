# 🏯 /win [decision] - WIN-WIN-WIN Validator

One command: Decision → Ngũ Sự Analysis → GO/NO-GO

## Arguments

- `$DECISION` - Decision or opportunity to evaluate

## What Runs (Silently)

### 1. Binh Pháp Ngũ Sự Analysis

```bash
PYTHONPATH=. python3 -c "
print('🏯 Analyzing with Ngũ Sự (5 Factors)...')
print()
print('ĐẠO (Alignment): Checking stakeholder alignment...')
print('THIÊN (Timing): Evaluating market timing...')
print('ĐỊA (Position): Assessing competitive position...')
print('TƯỚNG (Leadership): Reviewing team capability...')
print('PHÁP (Process): Validating execution process...')
"
```
