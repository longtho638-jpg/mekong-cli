## Usage

```bash
/refactor [action] [options]
```

## Actions/Options

| Action/Option | Description | Example |
|--------------|-------------|---------|
| `simplify` | Simplify complex logic | `/refactor simplify "complex_algo.py"` |
| `modernize` | Update syntax/patterns | `/refactor modernize "legacy_api.ts"` |
| `clean` | Clean up dead code | `/refactor clean "src/utils"` |
| `--dry-run` | Preview changes | `/refactor simplify --dry-run` |

## Execution Protocol

1. **Agent**: Delegates to `refactoring-specialist` (or `code-simplifier`).
2. **Process**:
   - Analyzes AST/Complexity.
   - Applies clean code principles (DRY, KISS).
   - Runs tests to ensure no regression.
3. **Output**: Refactored Code.

## Examples
