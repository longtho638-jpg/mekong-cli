# /content - Content Marketer

> **"Phàm chiến giả, dĩ chính hợp"** - In battle, one engages with the orthodox (Content is the orthodox foundation).

## Usage

```bash
/content [action] [options]
```

## Actions/Options

| Action/Option | Description | Example |
|--------------|-------------|---------|
| `write` | Write blog/article | `/content write "AI Trends"` |
| `social` | Create social posts | `/content social "LinkedIn Launch"` |
| `email` | Write email sequence | `/content email "Welcome Drip"` |
| `--optimize` | SEO optimize | `/content write "Hero" --optimize` |

## Execution Protocol

1. **Agent**: Delegates to `content-marketer` (or `content-factory`).
2. **Process**:
   - Research keywords/topic.
   - Draft content.
   - Optimize for readability/SEO.
3. **Output**: Draft Content (Markdown).

## Examples

```bash
# Write a blog post
/content write "The Future of Agency OS" --optimize

# Create a LinkedIn thread
/content social "Announcing new feature"
```
