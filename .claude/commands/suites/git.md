## Mission

Unified git command hub.

## Subcommands

| Command | Description | Agent |
|---------|-------------|-------|
| `/git:cm` | Commit changes | `git-manager` |
| `/git:cp` | Commit + Push | `git-manager` |
| `/git:pr` | Create PR | `git-manager` |

## Quick Examples

```bash
/git                  # Show git status
/git:cm               # Commit with message
/git:cp               # Commit and push
/git:pr main          # Create PR to main
```
