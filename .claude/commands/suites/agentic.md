## Mission

Unified view of the entire agentic AI workforce.

## Subcommands

| Command | Description |
|---------|-------------|
| `/agentic` | Full dashboard |
| `/agentic:stats` | Quick stats |
| `/agentic:agents` | Agent inventory |
| `/agentic:crews` | Crew overview |
| `/agentic:skills` | Skill matrix |
| `/agentic:level` | Coding level |

## Quick Examples

```bash
/agentic              # Full dashboard
/agentic:stats        # Quick numbers
/agentic:crews        # List crews
/agentic:level 5      # Set to GOD mode
```
