## THE 80/20 OPERATING MODEL

```
OpenClaw (80% of work):
  ✓ Writes all content (blog, email, social, docs)
  ✓ Builds and fixes all code
  ✓ Analyzes all data, produces all reports
  ✓ Manages all customer communication drafts
  ✓ Executes all workflows (onboard, upsell, bug-fix)
  ✓ Runs all marketing campaigns (draft → schedule)
  ✓ Monitors all metrics, flags anomalies
  ✓ Manages hiring pipeline (JDs, screening, scheduling)
  ✓ Tracks all credits, reconciles billing
  ✓ Generates weekly briefs, investor updates

Human (20% of work):
  ✓ Approves before send (emails, deploys, hiring)
  ✓ Closes deals (final investor/customer calls)
  ✓ Sets strategic direction (/founder roadmap → approve)
  ✓ Handles legal signing
  ✓ Builds relationships (cannot be automated)
  ✓ Reviews /founder week → sets 3 priorities
```
