## Intel Overview

Business intelligence cho startups & agencies.

## Subcommands

| Command | Description | Output |
|---------|-------------|--------|
| `/intel` | Intel overview | Dashboard |
| `/intel/vc` | VC intelligence | VC list |
| `/intel/competitor` | Competitor analysis | Report |
| `/fundraise` | Fundraising guide | Playbook |

## Usage

```
/intel                           # Intel dashboard
/intel/vc "Series A Vietnam"     # Find VCs
/intel/competitor "Notion"       # Analyze competitor
/fundraise "Seed round $500K"    # Fundraising strategy
```

## VC Intelligence

| Stage | Amount | Typical VCs |
|-------|--------|-------------|
| Pre-seed | $50-200K | Angels, micro VCs |
| Seed | $500K-2M | Seed funds |
| Series A | $5-15M | Institutional |
| Series B | $20-50M | Growth funds |

## Competitor Analysis Framework

1. **Product** - Features, pricing
2. **Marketing** - Channels, messaging
3. **Team** - Key hires, culture
4. **Funding** - Investors, runway
5. **Weaknesses** - Where to attack

## Fundraising Checklist

- [ ] Deck ready (10-15 slides)
- [ ] Data room organized
- [ ] Financial model
- [ ] Cap table clean
- [ ] References ready
- [ ] Target VC list (50+)
