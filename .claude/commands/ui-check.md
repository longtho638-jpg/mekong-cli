# /ui-check - UI Command

> **MCP Integration**: Routes to `ui_server`

## Usage

```bash
/ui-check
```

## Features

- Verifies `@agencyos/ui` version
- Lists available components
- Checks integration status

## MCP Tools

- `ui_server.check_ui_version`
- `ui_server.list_components`
