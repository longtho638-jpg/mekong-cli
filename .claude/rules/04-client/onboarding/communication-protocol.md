# Communication Protocol Rules

> **"Thông suốt như dòng chảy"** - Communication flows like water.

## 1. Channels
- **Slack/Discord**: Daily sync, quick questions (Async first).
- **Email**: Formal decisions, contracts, invoices.
- **Video Call**: Weekly sprint planning, demos, urgent blockers.

## 2. Response Time (SLA)
- **Urgent**: < 1 hour (PagerDuty/Phone).
- **High**: < 4 hours.
- **Normal**: < 24 hours.

## 3. Updates
- **Daily**: Standup (Text-based).
- **Weekly**: Progress Report (PDF/Notion).
- **Monthly**: Steering Committee.
