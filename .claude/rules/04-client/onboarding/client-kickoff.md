# Client Kickoff Rules

> **"Đầu xuôi đuôi lọt"** - A good start ensures success.

## 1. Preparation
- **Research**: Analyze client's domain, competitors, and current stack.
- **Team**: Assign Project Manager, Lead Dev, and Binh Phap Strategist.
- **Tools**: Setup Slack, Jira/Linear, and GitHub Repo.

## 2. The Kickoff Meeting
- **Agenda**:
    1. Introductions & Roles
    2. Vision & Goals Alignment (WIN-WIN-WIN)
    3. Communication Protocols
    4. Immediate Next Steps
- **Deliverables**: Kickoff Deck, Meeting Minutes, Access Request List.

## 3. Post-Kickoff
- **Access**: Verify access to all client systems within 24h.
- **Plan**: Finalize the first Sprint Plan.
