# Feedback Loop Rules

> **"Lắng nghe để thấu hiểu"** - Listen to understand.

## 1. Continuous Feedback
- **Weekly**: Sprint Review feedback.
- **Monthly**: Satisfaction check-in.
- **Ad-hoc**: After major milestones.

## 2. Handling Negative Feedback
- Acknowledge immediately.
- Investigate root cause (5 Whys).
- Propose corrective action plan.
- Follow up on resolution.

## 3. Turning Feedback into Gold
- Feature requests -> Product Roadmap.
- Pain points -> Process Improvements.
- Testimonials -> Marketing Assets.
