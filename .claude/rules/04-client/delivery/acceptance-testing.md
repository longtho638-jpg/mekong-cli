# Acceptance Testing Rules

> **"Khách hàng là trọng tài"** - Customer is the referee.

## 1. UAT Process
- Deploy to UAT/Staging environment.
- Provide test data and credentials.
- Walkthrough demo with client.

## 2. Feedback
- Log bugs in tracking system (Linear/Jira).
- Classify severity (Blocker, Critical, Major, Minor).
- Prioritize fixes for current or next sprint.

## 3. Sign-off
- Formal approval from Client PO.
- Trigger deployment to Production.
