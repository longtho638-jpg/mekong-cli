# Sprint Delivery Rules

> **"Quân quý thần tốc"** - Speed is essence of war.

## 1. Sprint Structure
- **Duration**: 2 weeks.
- **Planning**: Monday Week 1.
- **Review/Demo**: Friday Week 2.
- **Retrospective**: Friday Week 2 (Internal).

## 2. Definition of Done (DoD)
- Code implemented & reviewed.
- Tests passed (Unit & Integration).
- Documentation updated.
- Deployed to Staging.
- Feature Flag enabled (if applicable).

## 3. Deliverables
- Working software.
- Release notes.
- Updated metrics.
