# Data Privacy Rules

> **"Dữ liệu là tài sản"** - Data is asset (and liability).

## 1. Regulations
- HIPAA (US), GDPR (EU), CCPA (California).

## 2. Principles
- **Data Minimization**: Collect only what is needed.
- **Consent**: Explicit opt-in.
- **Right to be Forgotten**: Deletion capability.

## 3. Implementation
- Privacy Policy.
- Cookie Consent.
- Data Anonymization/Pseudonymization for analytics.
