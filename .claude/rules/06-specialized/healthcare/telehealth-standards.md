# Telehealth Standards Rules

> **"Khám chữa từ xa"** - Remote care.

## 1. Video/Audio Quality
- Low latency (<200ms).
- High definition (720p+).
- Fallback mechanisms for poor connection.

## 2. Usability
- One-click join (No install preferred).
- Mobile responsive.
- Accessibility (Captions, Screen readers).

## 3. Integration
- EHR/EMR integration (HL7/FHIR).
- ePrescribing.
- Appointment scheduling.
