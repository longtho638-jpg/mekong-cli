# Transaction Security Rules

> **"Giao dịch bất khả xâm phạm"** - Inviolable transactions.

## 1. Integrity
- Atomicity (ACID compliance).
- Idempotency keys (Prevent double charge).
- Audit trails (Immutable ledger).

## 2. Authentication
- SCA (Strong Customer Authentication).
- 3D Secure 2.0.
- Biometrics.

## 3. Fraud Detection
- Velocity checks.
- Geolocation checks.
- AI/ML anomaly detection.
