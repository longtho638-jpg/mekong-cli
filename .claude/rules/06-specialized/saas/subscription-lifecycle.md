# Subscription Lifecycle Rules

> **"Vòng đời thuê bao"** - Subscription lifecycle.

## 1. Stages
- **Trial**: Acquisition & Activation.
- **Active**: Value realization.
- **Past Due**: Retention effort.
- **Canceled**: Win-back / Offboarding.

## 2. Logic
- Proration (Upgrades/Downgrades).
- Grandfathering (Legacy pricing).
- Pausing/Resuming.

## 3. Metrics
- MRR/ARR movements.
- Churn analysis.
- Cohort analysis.
