### 1. Content Production

#### Video Production Pipeline
- **Pre-production:** Creative brief, script, storyboard, location scouting, casting, shot list
- **Production:** Camera setup, lighting (3-point, natural, cinematic), audio capture, directing talent
- **Post-production:** Editing (cut, color grade, sound mix), VFX, captions, export specs per platform
- **Distribution:** Platform optimization (YouTube, TikTok, Instagram, LinkedIn), thumbnail A/B test

#### Podcast Production
- **Format Design:** Solo, interview, panel, narrative, hybrid — match audience behavior
- **Recording:** DAW setup (Audacity/Descript/Riverside), mic technique, room treatment
- **Editing:** De-noise, EQ, compression, leveling (-16 LUFS for Spotify, -19 LUFS Apple)
- **Distribution:** RSS feed, Spotify for Podcasters, Apple Podcasts, Buzzsprout, Anchor
- **Monetization:** Dynamic ad insertion, host-read ads, Patreon, sponsorship rate card (CPM $18-$50)

#### Photography & Visual
- **Brand Photography:** Style guide alignment, shot list by use case (hero, lifestyle, product)
- **Product Shoots:** Flatlay, 3D render, 360 spin, white background, lifestyle context
- **Editing Workflows:** Lightroom batch presets, Photoshop retouching, export for web (WebP, AVIF)
- **Stock Strategy:** Unsplash, Getty, Shutterstock licensing tiers, AI-generated (Midjourney, DALL-E)

#### Motion Graphics & Animation
- **Explainer Videos:** Script → voiceover → animation (After Effects, Lottie, Rive)
- **Social Animations:** Reels/Shorts format (9:16, <30s), looping GIFs, Canva Pro motion
- **Brand Videos:** Manifesto, culture, product launch, testimonial compilation
- **Technical:** LottieFiles export, SVG animation, CSS keyframes, Framer Motion

### 2. Media Planning & Buying

```
MEDIA MIX FRAMEWORK:
  Awareness   → TV, YouTube, Podcast, Display (CPM $2-$25)
  Consideration→ Meta, LinkedIn, Pinterest, Native (CPC $0.5-$8)
  Conversion  → Search (Google/Bing), Retargeting, Shopping (CPA target)
  Retention   → Email, Push, SMS, Community (CLV optimization)
```

- **CPM/CPC/CPA Benchmarks:** Track by channel, creative, audience segment
- **Reach & Frequency:** Optimal freq 3-7x per flight to avoid ad fatigue
- **Media Mix Modeling (MMM):** Regression-based attribution across offline + online
- **Programmatic:** DSP (DV360, TTD, Xandr), PMP deals, open auction, header bidding

### 3. Digital Media Strategy

- **Social Media Strategy:**
  - Platform selection matrix: audience demographics x content format x brand fit
  - Content calendar (60/30/10 rule: edu/engage/promo)
  - Community management, social listening (Brandwatch, Sprout Social)
  - Influencer tiers: Nano (<10K), Micro (10-100K), Macro (100K-1M), Mega (1M+)
- **Content Marketing:**
  - Pillar + cluster model for SEO authority building
  - Repurposing framework: 1 long-form → 10+ micro-content pieces
  - Editorial calendar aligned to buyer journey
- **SEO/SEM:**
  - Keyword research (Search Vol, KD, CPC), on-page optimization, technical SEO
  - Google Ads: Smart Bidding, Performance Max, RSA, audience layering
  - Quality Score optimization: CTR, landing page relevance, expected CTR

### 4. Analytics & Performance

- **Engagement Metrics:** View rate, watch time %, completion rate, CTR, shares, saves
- **ROAS Calculation:** Revenue / Ad Spend — target varies by margin (3x for 30% margin)
- **Attribution Models:**
  - Last-click (default) → Linear → Time-decay → Data-driven (preferred)
  - GA4 cross-channel paths, Meta Attribution Window (1d/7d click, 1d view)
- **A/B Testing Protocol:**
  - Test 1 variable at a time (creative, copy, audience, placement)
  - Minimum 1000 impressions per variant before calling winner
  - Statistical significance: p-value <0.05, Bayesian preferred for small budgets

### 7. Creative Performance Optimization

- **Hook Formula:** First 3 seconds decide retention — pattern interrupt + bold claim + curiosity gap
- **Creative Fatigue Detection:** Frequency >5 + CTR decline >20% = swap creative
- **UGC vs Branded:** UGC converts 4x better for DTC, branded for enterprise trust
- **Copy Frameworks:** PAS (Problem-Agitate-Solve), AIDA, Before/After/Bridge
