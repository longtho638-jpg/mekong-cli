### 1. Fundraising & Development

#### Major Gifts
- **Prospect Identification:** Wealth screening, capacity + propensity + affinity scoring
- **Cultivation Ladder:** Suspect → Prospect → Cultivate → Ask → Steward
- **Gift Range Chart:** 10% donors = 60% revenue (apply Pareto principle)
- **Ask Formula:** Research → Relate → Request → Respond → Receive
- **Moves Management:** Track each touchpoint, assign relationship manager

#### Annual Fund
- **Segmentation:** LYBUNT, SYBUNT, New Donor, Lapsed (2-5yr), Major Gift prospect
- **Upgrade Strategies:** $250→$500 messaging, leadership giving circles
- **Retention Tactics:** Thank-you call within 48h, welcome series, anniversary recognition
- **Multi-channel:** Direct mail, email, phone, peer-to-peer, social

#### Capital Campaigns
- **Phases:** Pre-Campaign Feasibility → Silent Phase (60-70% of goal) → Public Launch → Countdown
- **Lead Gift Standard:** Top 5 gifts = 40-50% of total goal
- **Campaign Materials:** Case for Support, naming opportunities, pledge agreement, recognition plan
- **Volunteer Leadership:** Campaign Cabinet, co-chairs, division chairs

#### Planned Giving
- **Vehicles:** Bequest, charitable remainder trust, charitable lead trust, gift annuity, DAF
- **Marketing:** Legacy Society, inclusion in estate plans, beneficiary designation
- **Cultivation:** Long-term relationship, no rush, focus on donor's values

#### Events
- **Event ROI Formula:** (Gross Revenue - Direct Costs) / Direct Costs
- **Auction Best Practices:** Mobile bidding, 30% live/70% silent, procurement value = 50% of goal
- **Peer-to-Peer:** Platform selection, fundraiser training, social sharing toolkit

### 2. Grant Management

#### Prospect Research
- **Sources:** Foundation Directory Online, Candid, IRS Form 990, funder websites, GuideStar
- **Alignment Check:** Geography, mission, population, program area, grant size range
- **LOI vs Full Proposal:** Check funder guidelines — LOI required for 70% of foundations

#### Proposal Writing Framework
```
EXECUTIVE SUMMARY    → Need + Solution + Impact + Request (1 page)
STATEMENT OF NEED    → Data-driven, local stats, compelling narrative
PROJECT DESCRIPTION  → Goals, objectives, activities, timeline, logic model
EVALUATION PLAN      → Process + outcome metrics, data collection methods
ORGANIZATIONAL INFO  → Mission, history, capacity, leadership, financials
BUDGET NARRATIVE     → Line-by-line justification, cost-share, sustainability
ATTACHMENTS          → IRS 501(c)(3) letter, board list, audited financials
```

#### Grant Reporting
- **Interim Reports:** Mid-point narrative + financials vs budget
- **Final Reports:** Outcomes achieved, lessons learned, budget reconciliation, media
- **Relationship Maintenance:** Funder site visit, program officer lunch, thank-you calls

#### Compliance & Budgeting
- **Restricted Fund Tracking:** Separate ledger codes per grant, no co-mingling
- **Allowable Costs:** Direct vs indirect, funder overhead caps (typically 10-20%)
- **Carryover Requests:** Submit 60 days before grant end if underspent

### 3. Program Design & Evaluation

#### Logic Model
```
INPUTS → ACTIVITIES → OUTPUTS → SHORT-TERM OUTCOMES → LONG-TERM OUTCOMES → IMPACT
(Resources)  (What we do)  (Units of service)  (Changes in knowledge/attitude)  (Changes in behavior)  (Community change)
```

#### Theory of Change
- **If-Then Chain:** If X inputs + Y activities, then Z outcomes, because [assumption]
- **Assumptions:** What must be true for the model to work
- **Evidence Base:** What research supports this approach

#### Outcomes Measurement
- **SMART Outcomes:** Specific, Measurable, Achievable, Relevant, Time-bound
- **Data Collection:** Pre/post surveys, case notes, attendance tracking, test scores
- **Baseline:** Establish year 1, track change year-over-year

### 4. Volunteer Management

- **Recruitment:** Role descriptions, skills-based volunteering, corporate partnerships
- **Onboarding:** Orientation, background check, role training, buddy system
- **Scheduling:** Volunteer management software (VolunteerHub, Galaxy Digital, SignUpGenius)
- **Recognition:** Annual appreciation event, milestone certificates, public acknowledgment
- **Retention Formula:** Match skills to tasks + meaningful work + appreciation = loyalty

### 5. Board Governance

- **Fiduciary Duties:** Duty of Care, Duty of Loyalty, Duty of Obedience
- **Board Composition:** 3Ws — Wealth, Work, Wisdom; diversity matrix
- **Committee Structure:** Executive, Finance/Audit, Governance/Nominating, Program, Development
- **Meeting Cadence:** Full Board (quarterly), Executive Committee (monthly), Committees (monthly)
- **Strategic Planning:** 3-year cycle, SWOT, environmental scan, stakeholder input, theory of change
- **Bylaws Review:** Every 3-5 years, legal counsel, IRS compliance check
- **Board Assessment:** Annual self-evaluation, individual performance feedback

### 6. Community Engagement

- **Stakeholder Mapping:** Power/Interest matrix — Manage Closely / Keep Satisfied / Keep Informed / Monitor
- **Partnerships:** MOU framework, shared outcomes, co-branding, resource sharing
- **Advocacy:** Grassroots vs grasstops, policy brief, legislator meetings, comment letters
- **Coalition Building:** Shared agenda, governance structure, communication protocol, decision rules

### 7. Impact Measurement

- **SROI (Social Return on Investment):** (Net Present Value of Benefits) / Investment
- **Beneficiary Tracking:** Unique ID system, longitudinal follow-up (6mo, 12mo, 24mo)
- **Data Collection Tools:** Salesforce NPSP, Apricot, Efforts to Outcomes, Bonterra
- **Reporting Frameworks:** GRI Standards, IRIS+, SDG alignment, United Way CLA

### 8. Nonprofit Marketing & Communications

- **Storytelling Formula:** Hero (beneficiary) + Challenge + Transformation + Call to Action
- **Donor Communications:** Acknowledgment letter (48h), impact report (quarterly), newsletter (monthly), annual report (annually)
- **Annual Report:** Must-haves: financials, program stats, donor list, board list, mission statement
- **Digital Engagement:** Email (MailChimp/Constant Contact), social (FB/IG/LinkedIn), website donation page optimization
- **Matching Gift Campaigns:** Double the Donation integration, employer match promotion
