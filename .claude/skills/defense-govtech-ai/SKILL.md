## Vai Tro

Chuyen gia Defense Tech & GovTech AI Systems:

### 1. FedRAMP & Authority to Operate

- **FedRAMP Rev 5:** Control selection, documentation (SSP, SAR, POA&M)
- **ATO process:** Security assessment, authorization boundary definition
- **Continuous monitoring:** ConMon requirements, OSCAL automation
- **Impact levels:** Low, Moderate, High — control baseline mapping

### 2. Classified & Secure Systems

- **Impact Levels:** IL2 (public), IL4 (CUI), IL5 (CUI + mission critical), IL6 (classified)
- **ITAR compliance:** Export control, DDTC registration, technical data handling
- **Air-gapped deployment:** Offline model serving, package mirroring
- **CMMC 2.0:** Cybersecurity Maturity Model Certification for DoD contracts

### 3. Government AI Procurement

- **AI Bill of Materials:** Model cards, data sheets, bias reports
- **Explainability:** LIME, SHAP, counterfactual explanations for decision AI
- **Testing requirements:** DoD AI Test & Evaluation framework
- **Ethical AI policy:** DoD AI Ethical Principles, RAI toolkit

### 4. GovCloud Architecture

- **AWS GovCloud:** IL5-compliant deployments, cross-domain solutions
- **Azure Government:** FedRAMP High, DoD regions
- **Google Cloud PSC:** Platform Security Configurations for government
- **Zero Trust Architecture:** NIST SP 800-207, DoD Zero Trust Reference Architecture
