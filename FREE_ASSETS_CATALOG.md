```bash

git clone https://github.com/longtho638-jpg/mekong-cli.git

# Symlink to your project
```

### Troubleshooting

#### Issue: Agents not found
**Solution:**
```bash
# Verify .claude structure
ls -la .claude/agents/

# Ensure CLAUDE.md references agents
grep "@.claude/agents" CLAUDE.md

# Check symlinks
ls -l .claude/agents
```

#### Issue: Skills not loading
**Solution:**
```bash
# Check symlink
ls -l .claude-skills

# Re-create symlink
rm .claude-skills
ln -s /path/to/mekong-cli/.claude-skills .claude-skills

# Verify skill files exist
ls .claude-skills/
```

#### Issue: Commands not executing
**Solution:**
```bash
# Check command files
ls .claude/commands/

# Verify format
cat .claude/commands/cook.md

# Should contain frontmatter:
# ---
# name: cook
# description: Build feature autonomously
# ---
```
