## Troubleshooting

**Problem:** ZIP file upload fails
- **Solution:** Check file size (<100MB for Gumroad free tier)
- **Workaround:** Host large files on external storage, provide link

**Problem:** Description formatting broken
- **Solution:** Use Gumroad's Markdown editor
- **Avoid:** Complex HTML, use simple Markdown instead

**Problem:** Product not appearing in search
- **Solution:** Add more tags, improve SEO in description
- **Wait:** Gumroad indexing can take 24-48 hours
