# Project Kickoff Checklist

**Objective:** Successfully launch a new client project.
**Agent:** `product-manager`

## 1. Pre-Kickoff Setup
- [ ] **Contract Signed:** Verified in `03_Sales_Hub/Contracts/Signed/`.
- [ ] **Deposit Paid:** Verified in `04_Finance_Ops/Invoices/`.
- [ ] **Folder Created:** `01_Product_Factory/Projects/[Client_Name]`.
- [ ] **Team Assigned:** [List Team Members].

## 2. Information Gathering
- [ ] Client Intake Form received.
- [ ] Access credentials (WP, Hosting, Socials) secured in 1Password.
- [ ] Brand Assets (Logo, Fonts) downloaded to `Assets/`.

## 3. Internal Huddle
- [ ] Review Scope of Work (SOW).
- [ ] Identify Risks/Blockers.
- [ ] Set Internal Timeline.

## 4. Client Kickoff Call
- **Agenda:**
  1. Introductions.
  2. Review Goals & Success Criteria.
  3. Review Timeline & Milestones.
  4. Communication Protocol (Slack/Email).
  5. Next Steps.

## 5. Post-Kickoff
- [ ] Send Meeting Minutes.
- [ ] Update Project Management Tool (Linear/Trello/ClickUp).
- [ ] Schedule First Sprint.
