Feedback Widget Kit Demo

This page demonstrates how the Feedback Widget looks and feels. Look for the floating button in the bottom-right corner.

To integrate this into your React app, you would use code similar to this:

import { FeedbackWidget } from './components/FeedbackWidget';

function App() {
  return (
    <div>
      <YourApp />
      <FeedbackWidget
        apiEndpoint="http://localhost:8000/api/v1/feedback"
        primaryColor="#2563eb"
      />
    </div>
  );
}
