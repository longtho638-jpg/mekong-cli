## Setup Instructions (For Owner)

### Step 1: Enable Affiliate Program in Gumroad

1. Go to https://gumroad.com/settings/affiliates
2. Enable "Affiliates" for your account
3. Set commission rate to **20%**
4. Set cookie duration to **30 days**

### Step 2: Enable Per Product

For each product:

1. Edit product ‚Üí Settings ‚Üí Affiliates
2. Check "Enable affiliates for this product"
3. Save
