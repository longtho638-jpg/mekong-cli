## SOP-H01: Team Structure

```
Solo Phase (now):     CEO = all roles, AI bot = execution
Growth Phase:         Hire Quant Analyst → DevOps → Risk Manager
Scale Phase:          Full team with roles from CEO SOP-C06
```

## SOP-H02: Contributor Onboarding

```
1. Fork repo → clone → pnpm install
2. Read CLAUDE.md + docs/system-architecture.md
3. Run tsc --noEmit + pnpm test (must PASS)
4. Pick "good first issue" from GitHub
5. Submit PR → code review → merge
6. Join Discord #contributors channel
```

## SOP-H03: Hiring Triggers

| Role | Trigger | Priority |

| Quant Analyst | Revenue >$5K/mo | P1 |
| DevOps | 3+ exchanges live | P2 |
