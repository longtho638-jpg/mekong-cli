AGI Trader API

RaaS (Robot-as-a-Service) REST API for multi-tenant algorithmic trading. Manage strategies, execute trades, run backtests, and stream real-time data.

Base URLhttps://api.agitrader.io/v3

Authentication

All requests require a Bearer token. Obtain one via tenant creation or the /auth/token endpoint.

Authorization: Bearer at_live_xxxxxxxxxxxxxxxx

Tenants

POST/tenantsCreate a new tenant accountAuth

Request Body

namestringTenant display namerequired
planenumfree | pro | enterpriserequired
exchangesstring[]Exchange IDs to connect

Example

// Request POST /v3/tenants { "name": "Acme Capital", "plan": "pro", "exchanges": ["binance", "okx"] } // Response 201 { "id": "tn_abc123", "apiKey": "at_live_xxxx", "status": "active" }

Strategies

POST/strategies/deployDeploy a strategy to your tenantAuth
strategyIdstringStrategy ID from marketplacerequired
pairstringTrading pair (e.g. BTC/USDT)required
exchangestringTarget exchange IDrequired
configobjectStrategy-specific parameters
paperModebooleanPaper trading mode (default: true)
POST /v3/strategies/deploy { "strategyId": "agi-arb-v2", "pair": "BTC/USDT", "exchange": "binance", "paperMode": false, "config": { "scoreThreshold": 65 } }

Trades

GET/trades/positionsList open positions for tenantAuth
// Response 200 [{ "pair": "BTC/USDT", "side": "long", "size": 0.15, "entryPrice": 67240, "currentPrice": 67890, "unrealizedPnl": 97.50, "strategy": "agi-arb-v2" }]

Backtests

POST/backtestsRun a backtest with historical dataAuth
strategyIdstringStrategy to backtestrequired
pairstringTrading pairrequired
daysnumberHistorical days (max 365)required
balancenumberStarting balance in USDT

WebSocket Streams

Real-time data via WebSocket. Connect to wss://api.agitrader.io/v3/ws

WS/ws/tradesReal-time trade execution events
// Subscribe { "action": "subscribe", "channel": "trades" } // Event { "type": "trade.executed", "pair": "BTC/USDT", "side": "buy", "price": 67240, "size": 0.15, "strategy": "agi-arb-v2" }
WS/ws/alertsStrategy signals, spread alerts, system events

Channels: signals, spreads, system, risk
