### Phase 1: SDK Recipe System (`packages/vibe/src/recipes/`)
Mỗi package cần 1 recipe file mô tả khả năng:

```yaml

name: vibe-fnb
industry: food-and-beverage
capabilities:
  - menu-management
  - order-system
  - table-reservation
  - kitchen-display
provides:
  - MenuComponent
  - OrderFlow
  - KitchenDashboard
requires: [vibe-ui, vibe-auth]
starter_prompt: "Restaurant/cafe management system with menu, orders, and kitchen display"
```
